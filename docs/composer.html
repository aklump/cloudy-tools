Enabling Installation w/Composer

Index

This is a WIP page.

  1. Add composer.json to your cloudy package.
  2. Create bin folder
  3. Create the symlink in bin to the entry script, refer to cloudypm.yml
  4. Add this to composer.json to the bin section
{
    "bin": [
        "bin/live_dev_porter",
        "bin/ldp"
    ]
}
  1. Add the following to .gitignore to prevent core dependencies: /cloudy/.gitignore /cloudy/cache/ /cloudy/php/composer* /cloudy/php/vendor/ /cloudy/php/vendor/*
  2. Except for the vendor line, merge cloudy/.gitignore up, into your package's.
  3. An your package root type cloudy core.
  4. Change the path to cloudy in your controller file from to.
  5. Move all require from cloudy/composer.json to your package.
  6. Delete cloudy/composer.
  7. Edit cloudy/php/bootstrap.php autoload to require_once __DIR__ . '/../../vendor/autoload.php';
  8. composer require aklump/live-dev-porter:@dev
